## 表单

### 表单标签
* form
* input
* label 说明标签，for属性值与input 的id值一致进行关联
* fieldset 表单元素编组
* legend fieldset元素的说明标签
* button
* datalist 数据列表，包含option子标签，常与input一起使用（用户可不使用option内容）
* textarea
* select ,与option一起使用，也可联合optgroup 使用
